Proprietà associativa

Dati tre numeri \({\rm A}, {\rm B}, {\rm C}\) (che siano o meno interi) $$\begin{split} {\rm A} + ({\rm B} + {\rm C}) &= ({\rm A} + {\rm B}) + {\rm C}\\ {\rm A} \cdot ({\rm B} \cdot {\rm C}) &= ({\rm A} \cdot {\rm B}) \cdot {\rm C}. \end{split}$$

Esercizi:

  • Mostra che sono uguali: $$\begin{split} a + (b + (c + d)) &= (a + b) + (c + d)\\ (a + b) + (c + d) &= ((a + b) + c) + d\\ ((a + b) + c) + d &= (a + (b + c)) + d\\ (a + (b + c)) + d &= a + ((b + c) + d)\\ a + ((b + c) + d) &= a + (b + (c + d)). \end{split}$$
    Soluzione:
    $$\begin{split} {\rm A}, {\rm B}, {\rm C} &= a, b, c + d\\ {\rm A}, {\rm B}, {\rm C} &= a + b, c, d\\ {\rm A}, {\rm B}, {\rm C} &= a, b, c\\ {\rm A}, {\rm B}, {\rm C} &= a, b + c, d\\ {\rm A}, {\rm B}, {\rm C} &= b, c, d. \end{split}$$
  • Mostra che sono uguali: $$\begin{split} a \cdot (b \cdot (c \cdot d)) &= (a \cdot b) \cdot (c \cdot d)\\ (a \cdot b) \cdot (c \cdot d) &= ((a \cdot b) \cdot c) \cdot d\\ ((a \cdot b) \cdot c) \cdot d &= (a \cdot (b \cdot c)) \cdot d\\ (a \cdot (b \cdot c)) \cdot d &= a \cdot ((b \cdot c) \cdot d)\\ a \cdot ((b \cdot c) \cdot d) &= a \cdot (b \cdot (c \cdot d)). \end{split}$$
    Soluzione:
    $$\begin{split} {\rm A}, {\rm B}, {\rm C} &= a, b, c \cdot d\\ {\rm A}, {\rm B}, {\rm C} &= a \cdot b, c, d\\ {\rm A}, {\rm B}, {\rm C} &= a, b, c\\ {\rm A}, {\rm B}, {\rm C} &= a, b \cdot c, d\\ {\rm A}, {\rm B}, {\rm C} &= b, c, d. \end{split}$$

Proprietà commutativa

Dati due numeri \({\rm A}, {\rm B}\) (che siano o meno interi) $$\begin{split} {\rm A} + {\rm B} &= {\rm B} + {\rm A}\\ {\rm A} \cdot {\rm B} &= {\rm B} \cdot {\rm A}. \end{split}$$

Esercizi:

  • Mostra che sono uguali: $$\begin{split} a + b + c &= a + c + b\\ a + b + c &= b + a + c\\ a + b + c &= b + c + a\\ a + b + c &= c + a + b. \end{split}$$
    Soluzione:
    $$\begin{split} {\rm A}, {\rm B} &= b, c\\ {\rm A}, {\rm B} &= a, b\\ {\rm A}, {\rm B} &= a, b + c\\ {\rm A}, {\rm B} &= a + b, c. \end{split}$$
  • Mostra che sono uguali: $$\begin{split} a \cdot b \cdot c &= a \cdot c \cdot b\\ a \cdot b \cdot c &= c \cdot a \cdot b\\ a \cdot b \cdot c &= b \cdot a \cdot c\\ a \cdot b \cdot c &= b \cdot c \cdot a. \end{split}$$
    Soluzione:
    $$\begin{split} {\rm A}, {\rm B} &= b, c\\ {\rm A}, {\rm B} &= a, b\\ {\rm A}, {\rm B} &= a, b \cdot c\\ {\rm A}, {\rm B} &= a \cdot b, c. \end{split}$$

Proprietà distributiva

Dati tre numeri \({\rm A}, {\rm B}, {\rm C}\) (che siano o meno interi) $$\begin{split} {\rm A} \cdot ({\rm B} + {\rm C}) &= {\rm A} \cdot {\rm B} + {\rm A} \cdot {\rm C}. \end{split}$$

Esercizi:

  • Mostra che: $$\begin{split} a \cdot (b + c + d) &= a \cdot b + a \cdot c + a \cdot d. \end{split}$$
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow a + c = b + c\\ a = b &\longleftrightarrow a \cdot c = b \cdot c\\ a < b &\longleftrightarrow a + c < b + c\\ a < b &\longleftrightarrow a \cdot c < b \cdot c. \end{split}$$
    • La seconda vale se \(c \ne 0\).
    • La quarta vale se \(c > 0\).

Proprietà delle potenze

Dati tre numeri \({\rm A}, {\rm B}, {\rm C}\) (che siano o meno interi) $$\begin{split} {\rm A}^{{\rm B} + {\rm C}} &= {\rm A}^{\rm B} \cdot {\rm A}^{\rm C}\\ \left({\rm A}^{\rm B}\right)^{\rm C} &= {\rm A}^{{\rm B} \cdot {\rm C}}\\ ({\rm A} \cdot {\rm B})^{\rm C} &= {\rm A}^{\rm C} \cdot {\rm B}^{\rm C}. \end{split}$$

Esercizi:

  • Mostra che: $$\begin{split} a^{b + c + d} &= a^b \cdot a^c \cdot a^d\\ \left(\left(a^b\right)^c\right)^d &= a^{b \cdot c \cdot d}\\ (a \cdot b \cdot c)^d &= a^d \cdot b^d \cdot c^d. \end{split}$$
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow a^c = b^c\\ a = b &\longleftrightarrow c^a = c^b\\ a < b &\longleftrightarrow a^c < b^c\\ a < b &\longleftrightarrow c^a < c^b. \end{split}$$
    • La prima vale se \(c \ne 0\).
    • La seconda vale se \(c \ne 1\).
    • La terza vale se \(c > 0\).
    • La quarta vale se \(c > 1\).

Differenza e Divisione

La differenza e la divisione sono le inverse di somma e moltiplicazione, per cui sono definite come segue: $$\begin{split} {\rm A} - {\rm B} = {\rm C} &\longleftrightarrow {\rm A} = {\rm B} + {\rm C}\\ {\rm A} / {\rm B} = {\rm C} &\longleftrightarrow {\rm A} = {\rm B} \cdot {\rm C}. \end{split}$$ Per cui, ad esempio $$\begin{split} {\rm A} = {\rm B} + ({\rm A} - {\rm B}) &\qquad {\rm C} = ({\rm B} + {\rm C}) - {\rm B}\\ {\rm A} = {\rm B} \cdot ({\rm A} / {\rm B}) &\qquad {\rm C} = ({\rm B} \cdot {\rm C}) / {\rm B}. \end{split}$$

Esercizi:

  • Mostra che: $$\begin{split} (a - c) + b = (a + b) - c\\ a + (b - c) = (a + b) - c\\ a - (c - b) = (a + b) - c\\ (a - b) - c = a - (b + c)\\ (a / c) \cdot b = (a \cdot b) / c\\ a \cdot (b / c) = (a \cdot b) / c\\ a / (c / b) = (a \cdot b) / c\\ (a / b) / c = a / (b \cdot c)\\ (a - b) \cdot c = a \cdot c - b \cdot c\\ (a + b) / c = (a / c) + (b / c)\\ (a - b) / c = (a / c) - (b / c). \end{split}$$
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow a - c = b - c\\ a = b &\longleftrightarrow c - a = c - b\\ a < b &\longleftrightarrow a - c < b - c\\ a < b &\longleftrightarrow c - a < c - b. \end{split}$$
    • La quarta vale al contrario.
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow a / c = b / c\\ a = b &\longleftrightarrow c / a = c / b\\ a < b &\longleftrightarrow a / c < b / c\\ a < b &\longleftrightarrow c / a < c / b. \end{split}$$
    • La terza vale per \(c > 0\).
    • La quarta vale per \(c < 0\).

Radicali e Logaritmi

$$\begin{split} \sqrt[b]{a} = c &\longleftrightarrow a = c^b\\ \log_b{a} = c &\longleftrightarrow a = b^c. \end{split}$$

Esercizi:

  • Mostra che: $$\begin{split} a^{b - c} = a^b / a^c\\ a^{b / c} = \sqrt[c]{(a^b)}\\ a^{b / c} = (\sqrt[c]{a})^b\\ \sqrt[c]{a \cdot b} = \sqrt[c]{a} \cdot \sqrt[c]{b}\\ (a / b)^c = a^c / b^c\\ \sqrt[c]{(a / b)} = \sqrt[c]{a} / \sqrt[c]{b.} \end{split}$$
  • Mostra che: $$\begin{split} \log_a(b \cdot c) = \log_a(b) + \log_a(c)\\ \log_a(b / c) = \log_a(b) - \log_a(c)\\ \log_a(b^c) = c \cdot \log_a(b)\\ \log_a(\sqrt[c]{b}) = \log_a(b) / c\\ \log_a(b) \cdot \log_b(c) = \log_a(c)\\ \log_a(b) = \log_c(b) / \log_c(a) \end{split}$$
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow \sqrt[c]{a} = \sqrt[c]{b}\\ a = b &\longleftrightarrow \sqrt[a]{c} = \sqrt[b]{c}\\ a < b &\longleftrightarrow \sqrt[c]{a} < \sqrt[c]{b}\\ a < b &\longleftrightarrow \sqrt[a]{c} < \sqrt[b]{c} \end{split}$$
    • La seconda vale per \(c \ne 1\).
    • La quarta vale per \(c > 1\).
  • Correggi l'errore: $$\begin{split} a = b &\longleftrightarrow \log_c(a) = \log_c(b)\\ a = b &\longleftrightarrow \log_a(c) = \log_b(c)\\ a < b &\longleftrightarrow \log_c(a) < \log_c(b)\\ a < b &\longleftrightarrow \log_a(c) < \log_b(c) \end{split}$$
    • La seconda vale per \(c \ne 1\).
    • La terza vale per \(c > 1\).
    • La quarta vale per \(c < 1\).
