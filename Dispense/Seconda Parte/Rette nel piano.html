I triangoli

  • Teorema di Pitagora

    Dato un triangolo rettangolo di cateti ed ipotenusa \(x, y, z\) $$\begin{split} 4 x y + (x - y)^2 &= 2 x y + z^2 \end{split}$$ da cui $$\begin{split} z^2 &= x^2 + y^2 \end{split}$$ applicando tale principio ad un vettore \(u = (x_u, y_u)\), il modulo $$\begin{split} |u| = \sqrt{x_u^2 + y_u^2} \end{split}$$ è la distanza tra i punti \({\rm A}\) e \(\rm B\).

Le Rette

  • Dati due punti \({\rm A} = (x_{\rm A}, y_{\rm A})\) e \({\rm B} = (x_{\rm B}, y_{\rm B})\), questi identificano la retta formata dai punti $$\begin{split} {\rm A} + t u = ((1 - t) x_{\rm A} + t x_{\rm B}, (1 - t) y_{\rm A} + t y_{\rm B}) = {\rm B} + (1 - t) u \end{split}$$ al variare di \(t\). Cioè quella che chiamiamo retta parametrica $$\begin{split} x (t) &= (1 - t) x_{\rm A} + t x_{\rm B}\\ y (t) &= (1 - t) y_{\rm A} + t y_{\rm B}. \end{split}$$ L'equazione cartesiana che identifica tale retta è una qualunque di: $$\begin{split} (x - x_{\rm A})(y_{\rm A} - y_{\rm B}) &= (x_{\rm A} - x_{\rm B})(y - y_{\rm A})\\ (x - x_{\rm A})(y - y_{\rm B}) &= (x - x_{\rm B})(y - y_{\rm A})\\ (x - x_{\rm B})(y_{\rm A} - y_{\rm B}) &= (x_{\rm A} - x_{\rm B})(y - y_{\rm B}) \end{split}$$ Che sarebbe a dire $$\begin{split} x (y_{\rm A} - y_{\rm B}) + (x_{\rm B} - x_{\rm A}) y + (x_{\rm A} y_{\rm B} - x_{\rm B} y_{\rm A}) = 0. \end{split}$$ In tal caso si ha $$\begin{split} \frac{x - x_{\rm A}}{x_{\rm B} - x_{\rm A}} = t = \frac{y - y_{\rm A}}{y_{\rm B} - y_{\rm A}} \qquad \frac{x - x_{\rm B}}{x_{\rm B} - x_{\rm A}} = t - 1 = \frac{y - y_{\rm B}}{y_{\rm B} - y_{\rm A}.} \end{split}$$

    • Data una retta di equazione $$\begin{split} b x - a y = b x_{\rm A} - a y_{\rm A}, \end{split}$$

  • Si dice coefficiente angolare \(m\) il valore \(-\frac{a}{b.}\)
  • Si dice intercetta sull'asse \(m\) \(y\) il valore \(-\frac{c}{b.}\)
  • Date due rette di coefficienti angolari \(m_0, m_1\), queste sono parallele se \(m_0 = m_1\) e sono ortogonali se \(m_0 m_1 + 1 = 0\). In particolare, data una retta $$\begin{split} a x + b y + c = 0 \end{split}$$ ed un punto \({\rm A}\), la retta ortogonale passante per \({\rm A}\) ha equazione $$\begin{split} b x - a y = b x_{\rm A} - a y_{\rm A}. \end{split}$$ L'intersezione di queste rette è quindi $$\begin{split} \left(\frac{b^2 x_{\rm A} - a b y_{\rm A} - a c}{a^2 + b^2}, \frac{a^2 y_{\rm A} - a b x_{\rm A} - b c}{a^2 + b^2}\right). \end{split}$$ La distanza della retta da \({\rm A}\) è dunque $$\begin{split} \sqrt{\left(\frac{a^2 x_{\rm A} + a b y_{\rm A} + a c}{a^2 + b^2}\right)^2 + \left(\frac{b^2 y_{\rm A} + a b x_{\rm A} + b c}{a^2 + b^2}\right)^2} = \frac{|a x_{\rm A} + b y_{\rm A} + c|}{\sqrt{a^2 + b^2.}} \end{split}$$
  • Si dice mediana il luogo dei punti equidistanti da due punti dati, questa è ortogonale alla retta passante per i punti in questione e passa per il loro punto medio. $$\begin{split} (x - x_0)^2 + (y - y_0)^2 = (x - x_1)^2 + (y - y_1)^2\\ 2 (x_0 - x_1) x + 2 (y_0 - y_1) y = x_0^2 + y_0^2 - x_1^2 - y_1^2. \end{split}$$ Come previsto \(m = - \frac{x_0 - x_1}{y_0 - y_1.}\)

    • Il circocentro di un triangolo:

  • Si dice bisettrice il luogo dei punti equidistanti da due rette, questa è l'unione di $$\begin{split} \frac{a_0 x + b_0 y + c_0}{\sqrt{a_0^2 + b_0^2}} - \frac{a_1 x + b_1 y + c_1}{\sqrt{a_1^2 + b_1^2}} &= 0\\ \frac{a_0 x + b_0 y + c_0}{\sqrt{a_0^2 + b_0^2}} + \frac{a_1 x + b_1 y + c_1}{\sqrt{a_1^2 + b_1^2}} &= 0. \end{split}$$ Insomma due rette concorrenti con le rette date, di coefficienti angolari $$\begin{split} m_- = - \frac{a_0 \sqrt{a_1^2 + b_1^2} - a_1 \sqrt{a_0^2 + b_0^2}}{b_0 \sqrt{a_1^2 + b_1^2} - b_1 \sqrt{a_0^2 + b_0^2}} \qquad m_+ = - \frac{a_0 \sqrt{a_1^2 + b_1^2} + a_1 \sqrt{a_0^2 + b_0^2}}{b_0 \sqrt{a_1^2 + b_1^2} + b_1 \sqrt{a_0^2 + b_0^2.}} \end{split}$$

Le trasformazioni

  • Consideriamo una trasformazione $$\begin{split} u (x, y) = a_u x + b_u y + c_u\\ v (x, y) = a_v x + b_v y + c_v, \end{split}$$ Questa identifica un cambio di variabili $$\begin{split} x &= x (u, v)\\ y &= y (u, v), \end{split}$$ dove: $$\begin{split} u &= a_u x (u, v) + b_u y (u, v) + c_u\\ v &= a_v x (u, v) + b_v y (u, v) + c_v. \end{split}$$ Per risolvere scriviamo $$\begin{split} a_v u &= a_v a_u x (u, v) + a_v b_u y (u, v) + a_v c_u\\ b_v u &= b_v a_u x (u, v) + b_v b_u y (u, v) + b_v c_u\\ a_u v &= a_u a_v x (u, v) + a_u b_v y (u, v) + a_u c_v\\ b_u v &= b_u a_v x (u, v) + b_u b_v y (u, v) + b_u c_v, \end{split}$$ e semplifichiamo i termini uguali $$\begin{split} b_v u - b_u v &= b_v a_u x (u, v) + b_v c_u - b_u a_v x (u, v) - b_u c_v\\ a_v u - a_u v &= a_v b_u y (u, v) + a_v c_u - a_u b_v y (u, v) - a_u c_v, \end{split}$$ che sarebbe $$\begin{split} x (u, v) &= \left(\frac{b_v}{b_v a_u - b_u a_v}\right) u - \left(\frac{b_u}{b_v a_u - b_u a_v}\right) v + \left(\frac{b_u c_v - b_v c_u}{b_v a_u - b_u a_v}\right)\\ y (u, v) &= \left(\frac{a_u}{b_v a_u - b_u a_v}\right) v - \left(\frac{a_v}{b_v a_u - b_u a_v}\right) u + \left(\frac{a_v c_u - a_u c_v}{b_v a_u - b_u a_v}\right). \end{split}$$
  • Data una retta di equazione $$\begin{split} a x + b y + c = 0. \end{split}$$ nelle variabili \(x, y\), si dice il push-forward della retta il luogo geometrico identificato dall'equazione: $$\begin{split} a x (u, v) + b y (u, v) + c = 0 \end{split}$$ nelle variabili \(u, v\).
  • Data una retta di equazione $$\begin{split} a u + b v + c = 0, \end{split}$$ nelle variabili \(u, v\), si dice il pull-back della retta il luogo geometrico identificato dall'equazione: $$\begin{split} a u (x, y) + b v (x, y) + c = 0 \end{split}$$ nelle variabili \(x, y\).
  • Data invece una retta parametrica $$\begin{split} x (t) &= a t + x_0\\ y (t) &= b t + y_0, \end{split}$$ il suo push-forward è $$\begin{split} u (t) &= u (a t + x_0, b t + y_0)\\ v (t) &= v (a t + x_0, b t + y_0). \end{split}$$
  • Data invece una retta parametrica $$\begin{split} u (t) &= a t + u_0\\ v (t) &= b t + v_0, \end{split}$$ il suo pull-back è $$\begin{split} x (t) &= x (a t + u_0, b t + v_0)\\ y (t) &= y (a t + u_0, b t + v_0). \end{split}$$
