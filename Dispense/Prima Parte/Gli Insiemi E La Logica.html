Tavole di verità

  • Scriviamo \(\lnot\) per intendere la negazione "non".
  • Scriviamo \(\land\) per intendere la congiunzione "e".
  • Scriviamo \(\lor\) per intendere la disgiunzione "o".
  • Di seguito scriviamo \(0\) per intendere "falso".
  • Di seguito scriviamo \(1\) per intendere "vero".
  • La negazione di una frase è vera se la frase è falsa e viceversa: $$\begin{split} \lnot 0 &= 1\\ \lnot 1 &= 0. \end{split}$$
  • La congiunzione di due frasi è vera solo se lo sono entrambe: $$\begin{split} 1 \land 1 &= 1\\ 1 \land 0 &= 0\\ 0 \land 1 &= 0\\ 0 \land 0 &= 0. \end{split}$$
  • La disgiunzione di due frasi è vera se almeno una delle due lo è: $$\begin{split} 1 \lor 1 &= 1\\ 1 \lor 0 &= 1\\ 0 \lor 1 &= 1\\ 0 \lor 0 &= 0. \end{split}$$

Esercizi:

    Dimostra ciascuna delle seguenti utilizzando le altre $$\begin{split} a &= \lnot (\lnot a)\\ a &= a \land a\\ a &= a \lor a\\ \lnot ((\lnot a) \lor (\lnot b)) &= a \land b\\ \lnot ((\lnot a) \land (\lnot b)) &= a \lor b\\ a \land (b \land c) &= (a \land b) \land c\\ a \lor (b \lor c) &= (a \lor b) \lor c\\ a \land (b \lor c) &= (a \land b) \lor (a \land c)\\ a \lor (b \land c) &= (a \lor b) \land (a \lor c). \end{split}$$
  • Usando la terza si ha $$\begin{split} \lnot a = (\lnot a) \lor (\lnot a), \end{split}$$ mentre usando la quarta, $$\begin{split} \lnot ((\lnot a) \lor (\lnot a)) = a \land a, \end{split}$$ usando la seconda si conclude.
    Usando la seconda si ha $$\begin{split} \lnot a = (\lnot a) \land (\lnot a), \end{split}$$ mentre usando la quinta, $$\begin{split} \lnot ((\lnot a) \land (\lnot a)) = a \lor a, \end{split}$$ usando la terza si conclude.
    Usando la prima $$\begin{split} a \land a = \lnot (\lnot ((\lnot (\lnot a)) \land (\lnot (\lnot a)))), \end{split}$$ mentre usando la quinta e poi la terza, $$\begin{split} \lnot ((\lnot (\lnot a)) \land (\lnot (\lnot a))) = (\lnot a) \lor (\lnot a) = \lnot a. \end{split}$$ Infine, usando la prima si conclude.
    Usando la prima $$\begin{split} a \lor a = \lnot (\lnot ((\lnot (\lnot a)) \lor (\lnot (\lnot a)))), \end{split}$$ mentre usando la quarta e poi la seconda, $$\begin{split} \lnot ((\lnot (\lnot a)) \lor (\lnot (\lnot a))) = (\lnot a) \land (\lnot a) = \lnot a. \end{split}$$ Infine, usando la prima, si conclude.
    Usando la prima $$\begin{split} a \land b = \lnot (\lnot ((\lnot (\lnot a)) \land (\lnot (\lnot b)))), \end{split}$$ mentre usando la quinta, $$\begin{split} \lnot ((\lnot (\lnot a)) \land (\lnot (\lnot b))) = (\lnot a) \lor (\lnot b). \end{split}$$
    Usando la prima $$\begin{split} a \lor b = \lnot (\lnot ((\lnot (\lnot a)) \lor (\lnot (\lnot b)))), \end{split}$$ mentre usando la quarta, $$\begin{split} \lnot ((\lnot (\lnot a)) \lor (\lnot (\lnot b))) = (\lnot a) \land (\lnot b). \end{split}$$
    Usando la prima, la quarta e la quinta $$\begin{split} a \land (b \land c) = \lnot ((\lnot a) \lor (\lnot (b \land c)))\\ \lnot (b \land c) = (\lnot b) \lor (\lnot c)\\ \lnot ((\lnot (a \land b)) \lor (\lnot c)) = (a \land b) \land c\\ (\lnot a) \lor (\lnot b) = \lnot (a \land b). \end{split}$$ invece usando la settima $$\begin{split} (\lnot a) \lor ((\lnot b) \lor (\lnot c)) = ((\lnot a) \lor (\lnot b)) \lor (\lnot c). \end{split}$$
    Usando la prima, la quarta e la quinta $$\begin{split} a \lor (b \lor c) = \lnot ((\lnot a) \land (\lnot (b \lor c)))\\ \lnot (b \lor c) = (\lnot b) \land (\lnot c)\\ \lnot ((\lnot (a \lor b)) \land (\lnot c)) = (a \lor b) \lor c\\ (\lnot a) \land (\lnot b) = \lnot (a \lor b). \end{split}$$ invece usando la sesta $$\begin{split} (\lnot a) \land ((\lnot b) \land (\lnot c)) = ((\lnot a) \land (\lnot b)) \land (\lnot c). \end{split}$$
    Usando la prima, la quarta e la quinta $$\begin{split} a \land (b \lor c) = \lnot ((\lnot a) \lor (\lnot (b \lor c)))\\ \lnot ((\lnot (a \land b)) \land (\lnot (a \land c))) = (a \land b) \lor (a \land c)\\ \lnot (b \lor c) = (\lnot b) \land (\lnot c)\\ (\lnot a) \lor (\lnot b) = \lnot (a \land b)\\ (\lnot a) \lor (\lnot c) = \lnot (a \land c). \end{split}$$ Invece usando la nona $$\begin{split} (\lnot a) \lor ((\lnot b) \land (\lnot c)) = ((\lnot a) \lor (\lnot b)) \land ((\lnot a) \lor (\lnot c)). \end{split}$$
    Usando la prima, la quarta e la quinta $$\begin{split} a \lor (b \land c) = \lnot ((\lnot a) \land (\lnot (b \land c)))\\ \lnot ((\lnot (a \lor b)) \lor (\lnot (a \lor c))) = (a \lor b) \land (a \lor c)\\ \lnot (b \land c) = (\lnot b) \lor (\lnot c)\\ (\lnot a) \land (\lnot b) = \lnot (a \lor b)\\ (\lnot a) \land (\lnot c) = \lnot (a \lor c). \end{split}$$ Invece usando la ottava $$\begin{split} (\lnot a) \land ((\lnot b) \lor (\lnot c)) = ((\lnot a) \land (\lnot b)) \lor ((\lnot a) \land (\lnot c)). \end{split}$$

Quantificatori

  • Se \({\mathcal C}\) è un predicato, si scrive \(\forall x, {\mathcal C} (x)\) per dire "per ogni \(x\) si ha \({\mathcal C} (x)\)".
  • Se \({\mathcal C}\) è un predicato, si scrive \(\exists x: {\mathcal C} (x)\) per dire "esiste \(x\) tale che \({\mathcal C} (x)\)".
  • Si scrive \(a \longrightarrow b\) per dire "se \(a\), allora \(b\)".
  • Si scrive \(a \longleftarrow b\) per dire "\(a\), solo se \(b\)".
  • Si scrive \(a \longleftrightarrow b\) per dire "\(a\) se e solo se \(b\)".
  • Si possono scrivere i quantificatori l'uno in termini dell'altro come segue: $$\begin{split} (\exists x\!: {\mathcal C}(x)) &\longleftrightarrow \lnot (\forall x\,, \lnot {\mathcal C} (x))\\ (\forall x\,, {\mathcal C}(x)) &\longleftrightarrow \lnot (\exists x\!: \lnot {\mathcal C} (x)) \end{split}$$
  • Notiamo che se \({\mathcal A}, {\mathcal B}\) sono predicati, $$\begin{split} (\forall x\,, {\mathcal A}(x) \land {\mathcal B}(x)) &\longleftrightarrow (\forall x\,, {\mathcal A}(x)) \land (\forall x\,, {\mathcal B}(x))\\ (\exists x\!: {\mathcal A}(x) \lor {\mathcal B}(x)) &\longleftrightarrow (\exists x\!: {\mathcal A}(x)) \lor (\exists x\!: {\mathcal B}(x)). \end{split}$$ Cioè \(\forall\) è come un'enorme congiunzione, mentre \(\exists\) è come un'enorme disgiunzione.
  • Valgono inoltre le proprietà: $$\begin{split} (\exists x\!: {\mathcal A}(x) \land {\mathcal B}(x)) &\longleftarrow (\exists x\!: {\mathcal A}(x)) \land (\forall x\,, {\mathcal B}(x))\\ (\forall x\,, {\mathcal A}(x) \lor {\mathcal B}(x)) &\longrightarrow (\forall x\,, {\mathcal A}(x)) \lor (\exists x\!: {\mathcal B}(x)). \end{split}$$

Insiemi

Si scrive \(a \in {\rm A}\) per dire che \(a\) appartiene ad \({\rm A}\) ed \(a \notin {\rm A}\) per dire che non vi appartiene. Siano \({\rm A}, {\rm B}\) due insiemi,
  • Definiamo rispettivamente differenza "\(\setminus\)", intersezione "\(\cap\)", unione "\(\cup\)", come segue: $$\begin{split} {\rm A} \setminus {\rm B} &= \{a: a \in {\rm A} \land a \notin {\rm B}\}\\ {\rm A} \cap {\rm B} &= \{a: a \in {\rm A} \land a \in {\rm B}\}\\ {\rm A} \cup {\rm B} &= \{a: a \in {\rm A} \lor a \in {\rm B}\}. \end{split}$$
  • Scriviamo \({\rm A} \subseteq {\rm B}\) e diciamo che "\({\rm A}\) è contenuto in \({\rm B}\)" se \(\forall a \in {\rm A}\,, a \in {\rm B}\), cioè che ogni elemento di \({\rm A}\) appartiene a \({\rm B}\), che è equivalente a ciascuna delle seguenti: $$\begin{split} {\rm A} \setminus {\rm B} &= \emptyset\\ {\rm A} \cap {\rm B} &= {\rm A}\\ {\rm A} \cup {\rm B} &= {\rm B}. \end{split}$$

Esercizi:

  • Mostra che: $$\begin{split} (a \setminus b) \setminus c &= (a \setminus b) \cap (a \setminus c)\\ a \setminus (b \cup c) &= (a \setminus b) \cap (a \setminus c)\\ a \cap (b \setminus c) &= (a \cap b) \setminus (a \cap c)\\ (a \cap b) \setminus c &= (a \cap b) \setminus (a \cap c)\\ a \setminus (b \setminus c) &= (a \cap c) \cup (a \setminus b)\\ a \setminus (b \cap c) &= (a \setminus b) \cup (a \setminus c)\\ (a \cup b) \setminus c &= (a \setminus c) \cup (b \setminus c)\\ a \cup (b \setminus c) &= (a \cup b) \setminus (c \setminus a). \end{split}$$
    $$\begin{split} (a \land (\lnot b)) \land (\lnot c) &= (a \land (\lnot b)) \land (a \land (\lnot c))\\ (a \land (\lnot (b \lor c))) &= (a \land (\lnot b)) \land (a \land (\lnot c))\\ (a \land (b \land (\lnot c))) &= (a \land b) \land (a \land (\lnot c))\\ (a \land b) \land (\lnot c) &= (a \land b) \land (a \land (\lnot c))\\ (a \land (\lnot (b \land (\lnot c)))) &= (a \land c) \lor (a \land (\lnot b))\\ (a \land (\lnot (b \land c))) &= (a \land (\lnot c)) \lor (a \land (\lnot b))\\ (a \lor b) \land (\lnot c) &= (a \land (\lnot c)) \lor (b \land (\lnot c))\\ a \lor (b \land (\lnot c)) &= (a \lor b) \land (\lnot (c \land (\lnot a))). \end{split}$$
  • Mostra che se \(a \subseteq b\), allora: $$\begin{split} a \cap c &\subseteq b \cap c\\ a \cup c &\subseteq b \cup c\\ a \setminus c &\subseteq b \setminus c\\ c \setminus b &\subseteq c \setminus a\\ \end{split}$$
  • Mostra che $$\begin{split} c \subseteq (a \cap b) &\longleftrightarrow (c \subseteq a) \land (c \subseteq b)\\ (a \cup b) \subseteq c &\longleftrightarrow (a \subseteq c) \land (b \subseteq c)\\ \end{split}$$
